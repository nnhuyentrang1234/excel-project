# Excel Analysis Project

## Overview
This project demonstrates data analysis and reporting using **Microsoft Excel**.  
The focus is on organizing data, applying analytical logic, and presenting insights in a clear and structured way.

## Objectives
- Analyze the dataset using Excel tools
- Create meaningful summaries and insights
- Practice building a clean and reusable Excel analysis file

## Tools & Techniques
- Microsoft Excel
- Excel formulas
- Data cleaning
- PivotTables
- Charts & basic visualization

## Project Structure
- `excel/`
  - `HuyenTrang_Excel_project.xlsx` – Main Excel analysis file
- `screenshots/` – Dashboard screenshots
- `README.md` – Project documentation

## How to Use
1. Download the Excel file from the `excel/` folder
2. Open it in Microsoft Excel
3. Review the analysis, formulas, and results

## Notes
- This project is for learning and portfolio purposes
- The dataset used does not contain sensitive or confidential information
